# 01 — Recipes Index

🔑 LiveKit recipes are production-ready reference implementations grouped by domain (voice, telephony, workflows, advanced LLM).

Source: https://docs.livekit.io/recipes/

## Categories

- **Quickstarts**: voice AI starters across SwiftUI, Next.js, Flutter, React Native, Android, Web, Python, Node.js. Includes warm-transfer baseline.
- **Advanced LLM**: chain-of-thought agents, context variables, user-interruption handling, content filtering, structured output.
- **Voice Processing**: push-to-talk, mid-call language switching, transcription, keyword detection, translation pipelines, audio toggling.
- **Telephony**: IVR navigation, auto-answer, warm transfer, SIP lifecycle, Twilio Voice integration.
- **Audio**: background audio, TTS provider comparison, playback orchestration.
- **Agent Workflows**: medical-office triage, personal shopping, restaurant ordering, drive-thru, front-desk automation.
- **Advanced Features**: Gemini vision assistance, Raspberry Pi robotics, avatar agents (Tavus, LemonSlice), RAG with LlamaIndex, LangChain integrations.

## When to use

Treat each recipe as a starting scaffold — clone, swap providers, then extend. Recipes assume `livekit-agents` (Python) or `@livekit/agents` (Node) plus client SDK of the target platform.
