# 03 — LiveKit Cloud

🔑 Same OSS server, but managed: global mesh, no participant cap, built-in inference + telephony, 99.99% uptime.

Source: https://docs.livekit.io/intro/cloud/

## What's Included
- Globally distributed mesh of LiveKit servers — low-latency audio/video/data.
- **Agent Builder** — design + test agents in-browser.
- **Managed agent hosting** — scaling and lifecycle handled.
- **Native inference** — bundled LLM/STT/TTS access.
- **Telephony** — provision phone numbers for PSTN calls.
- **Analytics** — production-grade observability dashboard.

## Self-Hosted vs Cloud
| Capability | Self-Hosted | Cloud |
|---|---|---|
| Realtime media | Yes | Yes |
| Agents framework | Yes | Yes |
| Agent Builder | No | Yes |
| Managed hosting | No | Yes |
| Built-in inference | No | Yes |
| Per-room participants | ~3,000 max | Unlimited |
| Uptime SLA | DIY | 99.99% |

## Portability
Cloud runs the same open-source server (on GitHub), same APIs, same SDKs. Migrate either direction without code rewrites.

## Notes
- "Zero operational overhead" — no server management.
- Redundant infrastructure backs the uptime guarantee.
