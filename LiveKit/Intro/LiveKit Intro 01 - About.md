# 01 — About LiveKit

🔑 LiveKit is an open-source framework + managed cloud for real-time voice, video, and physical AI agents over WebRTC.

Source: https://docs.livekit.io/intro/about/

## What It Is
Infrastructure for AI agents to interact with users through real-time audio, video, and data streams. Same APIs and capabilities whether self-hosted or on LiveKit Cloud.

## Core Components
- **LiveKit Server** — open-source WebRTC SFU (Selective Forwarding Unit). Handles signaling, NAT traversal, RTP routing, QoS.
- **Agents Framework** — high-level tooling for speech processing, turn-taking, and LLM integration.
- **SDKs** — Web, iOS, Android, Flutter, Unity, plus backend SDKs (Python, Node.js, Go, Rust, Ruby, Kotlin).
- **Integration Services** — Egress (record/stream out), Ingress (external media in), Telephony (SIP/PSTN), server APIs.

## Use Cases
| Domain | Example |
|---|---|
| Voice AI | Assistants, customer service |
| Video | Conferencing, livestreaming |
| Health | HIPAA-compliant telehealth |
| Robotics | Physical AI / embodied agents |

## Deployment
- **LiveKit Cloud** — fully managed, globally distributed.
- **Self-hosted** — open-source server, identical APIs.
